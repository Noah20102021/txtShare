txtShare is a simple way to share text via web or shell.

Push

Get

Use in your PowerShell/Terminal

Connect:

Windows (PowerShell):

irm "http://HOSTNAME/setup" | iex

Linux / macOS:

. <(curl -s http://HOSTNAME/setup)

Use:
  • Push data:
    txtPush <Key> "<Inhalt>"
    Example: txtPush note "Buy milk"
  • Push password protected:
    txtPush <Key> "<Inhalt>" "<Password>"
    Example: txtPush secret "Secred" "topsecret123"
  • Get data:
    txtGet <Key>
    or with password: txtGet <Key> "<Password>"
  • Get help:
    txtHelp
